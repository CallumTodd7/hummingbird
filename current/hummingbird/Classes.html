Classes

The following classes are available globally.

  • HBApplication

    Application class. Brings together all the components of Hummingbird together

    Create an HBApplication, setup your application middleware, encoders, routes etc and then call start to start the server and wait to wait until the server is stopped.

    let app = HBApplication()
app.middleware.add(MyMiddleware())
app.get("hello") { _ in
    return "hello"
}
app.start()
app.wait()

    Editing the application setup after calling start will produce undefined behaviour.
